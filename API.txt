КОРОТКИЙ ОПИС МОЖЛИВОСТЕЙ API

API (Application Programming Interface) — Прикладни?й програ?мний інтерфе?йс (інтерфейс програмування застосунків, інтерфейс прикладного програмування). Якщо говорити більш зрозумілою мовою, то API - це готовий код для спрощення життя програмісту. API створювався для того, щоб програміст міг полегшити задачу написання того чи іншого додатка завдяки використанню готового коду. Якщо розглядати API в загальному, то можна знайти дуже багато сервісів, які представляють рішення для розробки. Найвідоміший на сьогоднішній день - це сервіс code.google.com, що надає близько півсотні різноманітних API! Це і інтерфейс для створення Android-додатків, і різні API для роботи з AJAX, і різні API додатків, які можна легко підстроїти під свій лад.

Адже чи є сенс писати код своїми руками? Навіщо заново винаходити колесо?

Правда є одне застереження. Початківцям НЕ слід користуватися напівготовими рішеннями, так як в майбутньому вони не впораються з реальним завданням. Тому, якщо ви початківець-програміст, то не використовуйте напівфабрикати! Вчіться думати своєю головою, будувати різні алгоритми, щоб зрозуміти суть програмування. Так само кажу, вже звертаючись до всіх, що API - це не готові рішення, це середовище, інтерфейс для створення своїх проектів. Ви ж не їсте замороженну котлети з магазину? Ви спочатку її підсмажете! Ця аналогія дуже ясно відображає суть API.
